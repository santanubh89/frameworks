Run Kafka:
kafka-server-start.bat ../../config/server.properties

Create Topic:
kafka-topics.bat --create --zookeeper localhost:2181 --replication-factor 3 --partitions 3 --topic USER_TOPIC

Describe Topic:
kafka-topics.bat --describe --topic USER_TOPIC --bootstrap-server localhost:9092
Topic: USER_TOPIC       PartitionCount: 3       ReplicationFactor: 3    Configs: segment.bytes=1073741824
        Topic: USER_TOPIC       Partition: 0    Leader: 1       Replicas: 1,2,0 Isr: 1,2,0
        Topic: USER_TOPIC       Partition: 1    Leader: 2       Replicas: 2,0,1 Isr: 2,0,1
        Topic: USER_TOPIC       Partition: 2    Leader: 0       Replicas: 0,1,2 Isr: 0,1,2
		
Start Producer:
kafka-console-producer.bat --topic USER_TOPIC --bootstrap-server localhost:9092

Start Consumer:
kafka-console-consumer.bat --topic USER_TOPIC --bootstrap-server localhost:9092
